プラットフォームやブラウザーに応じてコンテンツの制御をおこないます。ios, android, blackberry, chrome, safari, firefox, operaを指定できます。

例

<div ons-if-platform="android">
  ...
</div>

属性

ons-if-platform opera, firefox, safari, chrome, ie, android, blackberry, ios, windowsから指定します。
