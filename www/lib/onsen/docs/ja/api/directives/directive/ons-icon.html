アイコンを表示するコンポーネントです。Font AwesomeもしくはIoniconsから選択できます。

例

<ons-icon icon="fa-twitter" size="20px" fixed-width="false" style="color: red"></ons-icon>

属性

icon アイコン名を指定します。`fa-`で始まるものはFont Awesomeとして、`ion-`で始まるものはIoniconsとして扱われます。使用できるアイコンはこちら: http://fontawesome.io/icons/　および　http://ionicons.com。
size アイコンのサイズを指定します。値は、lg, 2x, 3x, 4x, 5xもしくはピクセル単位で指定できます。
rotate アイコンを回転して表示します。90, 180, 270から指定できます。
flip アイコンを反転します。horizontalもしくはverticalを指定できます。
fixed-width 等幅にするかどうかを指定します。trueもしくはfalseを指定できます。デフォルトはfalseです。
spin アイコンを回転するかどうかを指定します。trueもしくはfalseを指定できます。
