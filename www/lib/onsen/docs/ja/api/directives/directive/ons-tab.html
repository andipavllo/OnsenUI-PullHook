タブバーに配置される各アイテムのコンポーネントです。それぞれのons-tabはページを表します。

例

<ons-tabbar>
  <ons-tab page="home.html" active="true">
    <ons-icon icon="ion-home"></ons-icon>
    <span style="font-size: 14px">Home</span>
  </ons-tab>
  <ons-tab page="fav.html" active="true">
    <ons-icon icon="ion-star"></ons-icon>
    <span style="font-size: 14px">Favorites</span>
  </ons-tab>
  <ons-tab page="settings.html" active="true">
    <ons-icon icon="ion-gear-a"></ons-icon>
    <span style="font-size: 14px">Settings</span>
  </ons-tab>
</ons-tabbar>

<ons-template id="home.html">
  ...
</ons-template>

<ons-template id="fav.html">
  ...
</ons-template>

<ons-template id="settings.html">
  ...
</ons-template>

属性

page ons-tabが参照するページへのURLを指定します。
icon アイコン名を指定します。ons-iconと同じアイコン名を指定できます。個別にアイコンをカスタマイズする場合は、background-imageなどのCSSスタイルを用いて指定できます。
active-icon アクティブの際のアイコン名を指定します。
label アイコン下に表示されるラベルを指定します。
active このタブアイテムをアクティブ状態にするかどうかを指定します。trueもしくはfalseを指定できます。
no-reload すでにアクティブになったタブを再びクリックするとページの再読み込みは発生しません。
persistent このタブで読み込んだページを永続化します。この属性があるとき、別のタブのページに切り替えても、読み込んだページのDOM要素は破棄されずに単に非表示になります。
