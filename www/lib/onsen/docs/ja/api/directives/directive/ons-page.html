ページ定義のためのコンポーネントです。このコンポーネントの内容はスクロールが許可されます。

例

<ons-page>
  <ons-toolbar>
    <div class="center">Title</div>
  </ons-toolbar>

  ...
</ons-page>

属性

var このページを参照するための変数名を指定します。
modifier スタイル定義をカスタマイズするための名前を指定します。
on-device-backbutton デバイスのバックボタンが押された時の挙動を設定できます。
ng-device-backbutton デバイスのバックボタンが押された時の挙動を設定できます。AngularJSのexpressionを指定できます。

メソッド

getDeviceBackButtonHandler() バックボタンハンドラを取得します。このメソッドはnullを返す場合があります。
