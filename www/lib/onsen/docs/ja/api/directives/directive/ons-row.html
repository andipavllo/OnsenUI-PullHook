グリッドシステムにて行を定義します。ons-colとともに使用し、コンポーネントの配置に使用します。

例

<ons-row>
  <ons-col width="50px"><ons-icon icon="fa-twitter"></ons-icon></ons-col>
  <ons-col>Text</ons-col>
</ons-row>

属性

align 縦に整列するために指定します。top、bottom、centerのいずれかを指定できます。
