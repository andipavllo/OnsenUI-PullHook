ナビゲーションで使用するツールバー用コンポーネントです。クラス名により、左、中央、右のコンテナを指定できます。

例

<ons-page>
  <ons-toolbar>
    <div class="left"><ons-back-button>Back</ons-back-button></div>
    <div class="center">Title</div>
    <div class="right">Label</div>
  </ons-toolbar>
</ons-page>

属性

inline ツールバーをインラインに置きます。スクロール領域内にそのまま表示されます。
modifier ツールバーの表現を指定します。
fixed-style 中央揃え要素は、デフォルトとしてAndroidでは左配置、iOSでは中央配置です。この属性を使用すると、要素はAndroidとiOSともに中央配置となります。
